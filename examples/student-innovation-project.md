# 示例：大创项目包装

## 用户输入

```text
使用琢玉工坊帮我包装一个大创项目。

项目名称：AI 电学实验辅导系统
技术栈：Next.js、Supabase、Python、OpenAI API
已有材料：代码仓库、3 张截图、简单 README、结题报告初稿
目标用途：校级大创结题答辩
限制条件：没有上线用户量，没有论文，没有软著，不能虚构这些成果。
```

## 期望输出方向

- 一句话定位：面向电学实验预习和答疑的辅助系统。
- 真实成果：账号体系、实验资料管理、AI 问答、历史记录、后台管理。
- 证据补齐：功能截图、数据库结构截图、接口测试记录、一次完整演示视频。
- 包装重点：教学辅助场景、工程实现完整度、迭代过程、可扩展性。
- 避免表述：不要写“已大规模应用”“显著提升成绩”“获得软件著作权”。
