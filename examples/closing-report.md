# 示例：结题报告整理

## 用户输入

```text
使用琢玉工坊帮我整理结题报告。

项目名称：基于深度学习的校园垃圾分类识别系统
项目类型：大创结题
已有材料：训练代码、模型结果截图、测试集样例、报告初稿
缺失材料：没有正式部署，没有用户调研
学校要求：包含研究背景、技术路线、成果、经费使用、问题与展望
```

## 推荐输出内容

- 报告目录和每章写作要点。
- 技术路线：数据采集、预处理、模型训练、识别界面、结果分析。
- 成果矩阵：模型文件、训练日志、测试截图、前端演示、报告初稿。
- 风险说明：数据规模有限，未进行真实校园部署。
- 展望：扩充数据集、移动端部署、加入人工纠错闭环。
