# 示例：Codex 报错诊断

## 用户输入

```text
使用琢玉工坊按 Codex 安全工作流诊断报错。

项目技术栈：Next.js + Supabase
报错：注册接口返回 500，控制台显示 user_id violates not-null constraint
复现步骤：点击注册，邮箱验证前写入 profiles 表失败
限制条件：不能重构认证模块，优先最小修改。
```

## 期望处理方式

- 先检查认证回调、profiles 插入逻辑、数据库约束和触发器。
- 判断是用户 ID 未生成、插入时机错误，还是 RLS/trigger 配置问题。
- 提出最小修改方案，不直接重构登录注册流程。
- 给出验证命令或手动验证步骤。
- 说明失败后如何回滚到原注册逻辑。
